# Cardápio Digital & KDS (Kitchen Display System)

Aplicação web completa para gestão de restaurantes, focada em performance e experiência mobile-first. 

Tecnologias e Ferramentas
- Linguagens: HTML, CSS, JavaScript.
- Metodologia: Mobile-First e UI/UX focado em alta demanda.
- Workflow AI: Desenvolvido com auxílio de ferramentas de IA (Claude, Gemini e Antigravity) para otimização de código e interface.

Funcionalidades
- Interface Cliente:** Pedidos em tempo real com layout responsivo.
- Interface Cozinha (KDS):** Gestão de status de pedidos para fluxo de produção.
- Interface Admin:** Controle total de estoque e edição de cardápio.

Como executar
Basta clonar o repositório e abrir os arquivos .html no seu navegador.
